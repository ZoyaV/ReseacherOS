# Тексты понятным языком

Скилл следит за тем, как звучат заголовки и описания в ResearchOS: на карте
дерева и в канбане человек должен понять смысл без документации и без
смешения языков в одной фразе.

## Зачем он нужен

Короткий ярлык на карте (обычно до восьми слов), подробности — в описании.
Сначала смысл своими словами, потом при необходимости сокращение в скобках,
например «доля кликов по показам (CTR)». Технические id и сырые метрики
в видимый текст не попадают.

## Как пользоваться

В установленном ResearchOS скилл подключается, когда агент правит карточки,
узлы дерева или другие тексты для человека. Агент следует `SKILL.md` в
`agents/skills/koi-prose-style/`; эталонные пары «хорошо / плохо» — в
`examples.md`.

## Состав пакета

- `SKILL.md` — правила и цикл ревью для агента
- `examples.md` — примеры удачных и слабых формулировок
- `manifest.yaml` — карточка для каталога Hub
