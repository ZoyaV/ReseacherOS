# Генерация статьи (NeurIPS)

Генерация NeurIPS-статьи (LaTeX → PDF) из очереди paper UI.

## Как пользоваться

В установленном ResearchOS скилл уже доступен агенту из `agents/skills/koi-paper/` (и зеркала `.cursor/skills/`).

На Hub эта страница — описание пакета. Агент следует файлу `SKILL.md` в каталоге скилла.

## Состав пакета

- `SKILL.md`
- `manifest.yaml`
