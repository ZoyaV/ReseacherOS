Глубокая суммаризация knowledge base по завершённым экспериментам.

## Как пользоваться

В установленном ResearchOS скилл уже доступен агенту из `agents/skills/koi-knowledge-curator/` (и зеркала `.cursor/skills/`).

На Hub эта страница — описание пакета. Агент следует файлу `SKILL.md` в каталоге скилла.
