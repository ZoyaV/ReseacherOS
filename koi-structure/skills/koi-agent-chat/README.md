# Agent chat (вопрос из UI)

Ответы на вопросы из панели «Спросить агента» в UI ResearchOS.

## Как пользоваться

В установленном ResearchOS скилл уже доступен агенту из `agents/skills/koi-agent-chat/` (и зеркала `.cursor/skills/`).

На Hub эта страница — описание пакета. Агент следует файлу `SKILL.md` в каталоге скилла.

## Состав пакета

- `SKILL.md`
- `manifest.yaml`
