Пошаговый grill перед экспериментом: гипотеза, setup, критерии done.

## Как пользоваться

В установленном ResearchOS скилл уже доступен агенту из `agents/skills/koi-grill-experiment/` (и зеркала `.cursor/skills/`).

На Hub эта страница — описание пакета. Агент следует файлу `SKILL.md` в каталоге скилла.
