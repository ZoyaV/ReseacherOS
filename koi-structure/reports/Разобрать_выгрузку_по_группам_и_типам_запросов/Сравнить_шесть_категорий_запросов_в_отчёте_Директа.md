# Сравнить шесть категорий запросов в отчёте Директа

Статус: выполнен  
Канбан: seg-targeting  
Зависимости: seg-pareto-groups → эта карточка → блок устранения (перераспределение бюджета)

---

## 1. Описание эксперимента и зачем

Помимо рекламных групп, отчёт Директа режет данные по категории таргетинга — типу поискового запроса. Нужно понять, не утекает ли бюджет в широкие и конкурентные запросы с завышенной ценой клика.

Цель: сравнить шесть категорий таргетинга по расходу, числу кликов, доле кликов (CTR) и цене клика (CPC).

Зачем в гипотезе: если «целевые запросы» дают лучшую отдачу, чем «широкие» и «с упоминанием конкурентов», есть основание ужесточать таргетинг в сценариях устранения.

Что не является целью:

- повторять разбивку по 25 группам;
- моделировать перенос 1 467 ₽ (карточка sim-broad-targeting);
- менять минус-фразы в интерфейсе Директа.

---

## 2. Основная метрика (финальный тест)

Сравнение целевых запросов с объединённым сегментом «широкие + конкурентные + альтернативные» по цене клика (CPC) и доле кликов (CTR).

### Определение

| Параметр | Значение |
|----------|----------|
| Метрика | цена клика (CPC); вторично — доля кликов (CTR) и доля бюджета |
| Категории | целевые, узкие, с упоминанием конкурентов, альтернативные, широкие, не определено |
| Ориентир целевых | ~41 403 ₽ (83% бюджета), CTR ~12,5%, CPC ~36,6 ₽ |
| Ориентир слабых трёх | широкие + конкуренты + альтернативные ≈ 2 934 ₽, 57 кликов, CPC ~51 ₽ |
| Протокол | выгрузка Centra Market — велосипеды, май 2024; колонка «Категория таргетинга» |
| Скрипт / артефакты | лок.: `datasets/bicycle.csv`; табл. A в §4.1 |

## 3. Постановки и подготовка

### 3.1 Агрегация по категориям таргетинга → §4

Кратко: сгруппировать строки по «Категория таргетинга», посчитать метрики; отдельно суммировать «слабую тройку» для сравнения с целевыми.

- Используемые данные: выгрузка Centra Market — велосипеды, май 2024; лок.: `datasets/bicycle.csv`
- Слабая тройка для сводки: «Широкие запросы», «Запросы с упоминанием конкурентов», «Альтернативные запросы»
- Статус: выполнен → [§4](#4-эксперимент-1--категории-таргетинга)

Подзадачи:

- [x] Агрегировать по каждой из шести категорий: показы, клики, расход, CTR, CPC, доля бюджета
- [x] Свести слабую тройку в одну строку «итого слабые» в табл. B (диагностика)
- [x] Заполнить табл. A; в §4.2 сравнить CPC целевых (~37 ₽) и слабых (~51 ₽)

#### Сбор данных

```bash
python3 -c "
import csv
from collections import defaultdict
d=defaultdict(lambda:{'i':0,'c':0,'s':0.0})
for r in csv.DictReader(open('datasets/bicycle.csv', encoding='utf-8')):
    k=r['Категория таргетинга']
    d[k]['i']+=float(r['Показы'] or 0); d[k]['c']+=float(r['Клики'] or 0); d[k]['s']+=float(r['Расход (руб.)'] or 0)
for k,v in sorted(d.items(), key=lambda x:-x[1]['s']):
    ctr=v['c']/v['i']*100 if v['i'] else 0
    cpc=v['s']/v['c'] if v['c'] else 0
    print(k, f'{v[\"s\"]:.0f}₽', f'CTR {ctr:.1f}%', f'CPC {cpc:.1f}₽')
"
```

### 4.2 Выводы

**Гипотеза о разрыве отдачи между целевыми и слабой тройкой подтверждается.** Целевые запросы дают CTR **12,53%** и CPC **36,64 ₽** — ориентиры карточки (12,5% и 36,6 ₽) совпадают. Объединённый сегмент «широкие + конкуренты + альтернативные» — **2 934 ₽** (5,9% бюджета), **57 кликов**, CTR **6,79%**, CPC **51,48 ₽** (ориентир ~51 ₽ подтверждён). Клик в слабых категориях обходится на **40%** дороже, CTR ниже на **5,7** п.п.

**По отдельным слабым категориям:** наибольший абсолютный расход у конкурентных запросов (**1 477 ₽**, CPC **56,8 ₽**), но даже вместе слабая тройка — лишь **~6%** бюджета. Основная масса денег (**82,8%**) уже в целевых; утечка не массовая, но каждый рубль в слабых категориях покупает заметно меньше кликов.

**Узкие запросы** (**10,4%** бюджета) — отдельный паттерн: CTR **8,71%** (ниже целевых), но CPC **29,1 ₽** — самый низкий среди категорий с заметным расходом; не входят в «слабую тройку» финального теста.

**«не определено»** — мало денег (**452 ₽**), аномально низкий CTR (**2,81%**); на вывод по таргетингу почти не влияет.

Итог: целевые запросы однозначно эффективнее слабой тройки по CPC и CTR. Абсолютный перекос бюджета в широкие/конкурентные запросы умеренный (~3 000 ₽ из 50 000), но ужесточение таргетинга и сценарий sim-broad-targeting (срезать половину расхода слабой тройки) обоснованы — каждый сэкономленный рубль можно переложить в целевые по CPC **~37 ₽** вместо **~51 ₽**.
