# Сравнить текст и картинку на телефоне в семи рекламных группах

Статус: выполнен  
Канбан: fmt-paired  
Зависимости: fmt-aggregate → эта карточка → блок устранения (политика форматов)

---

## 1. Описание эксперимента и зачем

Агрегат по всей выгрузке может скрывать разницу между рекламными группами. Нужно проверить, сохраняется ли провал картинки на телефоне внутри одних и тех же групп, где крутятся оба формата.

Цель: для семи рекламных групп с достаточным объёмом показов сравнить долю кликов (CTR) текстового и графического объявления на телефоне.

Зачем в гипотезе: если текст стабильно обгоняет картинку внутри групп, эффект вряд ли объясняется только разным составом аудитории между группами.

Что не является целью:

- повторять сводку по всей выгрузке (сделано в fmt-aggregate);
- статистические тесты значимости (достаточно описательного сравнения для учебного проекта);
- рекомендации по бюджету (карточки fmt-policy-off / fmt-policy-realloc).

---

## 2. Основная метрика (финальный тест)

В каждой из семи групп — разница в доле кликов (CTR) между текстом и картинкой на телефоне. Успех: в большинстве групп (≥5 из 7) текст даёт более высокую долю кликов.

#### Сбор данных

Используем тот же файл `datasets/bicycle.csv`. Новый collect не нужен.

```bash
python3 -c "
import csv
from collections import defaultdict
by_g = defaultdict(lambda: defaultdict(lambda: {'i':0,'c':0}))
for r in csv.DictReader(open('datasets/bicycle.csv', encoding='utf-8')):
    if r['Тип устройства'] != 'мобильные': continue
    g, f = r['Группа'], r['Формат']
    by_g[g][f]['i'] += float(r['Показы'] or 0)
    by_g[g][f]['c'] += float(r['Клики'] or 0)
for g, fmts in sorted(by_g.items()):
    if 'текстовый' in fmts and 'графический' in fmts:
        t, gr = fmts['текстовый'], fmts['графический']
        if t['i']>=20 and gr['i']>=20:
            tctr = t['c']/t['i']*100
            gctr = gr['c']/gr['i']*100
            print(f'{g}: текст {tctr:.1f}% ({int(t[\"c\"])}/{int(t[\"i\"])}), картинка {gctr:.1f}%')
"
```

### 4.2 Выводы

Во всех **семи** отобранных группах текстовое объявление на телефоне даёт более высокую долю кликов, чем графическое — критерий «≥5 из 7» выполнен с запасом. Разрыв варьируется от **1,0 п.п.** («Веломагазин») до **11,0 п.п.** («Велосипед Ангарск»); в шести группах из семи преимущество текста превышает 5 п.п.

Парное сравнение **подтверждает** вывод агрегата fmt-aggregate: слабость картинки на телефоне не сводится к тому, что графические объявления случайно попали в «плохие» группы. Внутри одних и тех же групп, на одной и той же аудитории, текст стабильно обгоняет картинку. Это усиливает гипотезу о несоответствии формата и устройства и открывает путь к карточкам устранения (fmt-policy-off, fmt-policy-realloc).
